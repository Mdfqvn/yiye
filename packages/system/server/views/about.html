关于我们

一叶书签是一个基于兴趣的内容聚合与分享应用。通过一叶书签，你可以与他人一起收集和分享你感兴趣的事物，并且发现更多你关心的内容。

一叶书签团队成立于2014年9月份。在团队绝大多数同学大学即将毕业的最后一年。我们希望通过我们的想法和努力，做一款有意思的产品。可能它远远无法改变世界，但真的会有人喜欢它，因为它而生活多了一份趣味。所以我们开始做一款产品,一个更容易让人们分享和发现自己感兴趣内容的应用。那便是一叶书签。
